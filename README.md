## Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Omer-KISAKOL/product-catalog.git
    cd product-catalog
    ```
2. **Install dependencies**:
    ```bash
    pnpm install
    ```
3. **Start the Project**:
    ```bash
    pnpm run dev
    ```
